# AI Meeting Summarizer

An AI-powered meeting summarization tool that converts conversations into concise summaries, key insights, and action items using NLP techniques.

## 🚀 Features
- Generate meeting summaries automatically
- Extract key points and action items
- Browser extension-based interface
- Backend processing with worker support

## 🛠️ Tech Stack
- Frontend: Browser Extension (JavaScript)
- Backend: Node.js
- AI/NLP Processing
- Worker for background tasks

## ⚙️ Installation

1. Clone the repository
2. Install dependencies:
   npm install
3. Start the backend:
   npm start
